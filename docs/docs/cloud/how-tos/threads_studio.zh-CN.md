# 管理线程 (Manage threads)

Studio 允许您从服务器查看 [线程](../../concepts/persistence.md#threads) 并编辑其状态。

## 查看线程 (View threads)

### 图模式 (Graph mode)

1. 在右侧窗格的顶部，选择下拉菜单以查看现有线程。
2. 选择所需的线程，线程历史记录将填充在页面的右侧。
3. 要创建新线程，请单击 `+ New Thread` 并 [提交运行](../how-tos/invoke_studio.md#graph-mode)。

要查看线程中更详细的信息，请将页面顶部的滑块向右拖动。要查看更少的信息，请将滑块向左拖动。此外，可以折叠或展开状态的各个回合、节点和键。

在 `Pretty` 和 `JSON` 模式之间切换以获得不同的渲染格式。

### 聊天模式 (Chat mode)

1. 在页面右侧窗格中查看所有线程。
2. 选择所需的线程，线程历史记录将填充在中心面板中。
3. 要创建新线程，请单击加号按钮并 [提交运行](../how-tos/invoke_studio.md#chat-mode)。

## 编辑线程历史记录 (Edit thread history)

### 图模式 (Graph mode)

要编辑线程的状态，请选择所需节点旁边的 "edit node state"（编辑节点状态）。按需编辑节点的输出，然后单击 "fork"（分叉）进行确认。这将从所选节点的检查点创建一个新的分叉运行。

如果您想从给定的检查点重新运行线程而不编辑状态，请单击 "Re-run from here"（从此处重新运行）。这将再次从选定的检查点创建一个新的分叉运行。这对于重新运行非特定于状态的更改（例如选定的助手）非常有用。

### 聊天模式 (Chat mode)

要编辑线程中的人类消息，请单击人类消息下方的编辑按钮。按需编辑消息并提交。这将创建对话历史记录的新分叉。要重新生成 AI 消息，请单击 AI 消息下方的重试图标。
