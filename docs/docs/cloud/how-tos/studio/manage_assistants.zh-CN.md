# 管理助手 (Manage assistants)

!!! info "Prerequisites"

    - [Assistants Overview](../../../concepts/assistants.md)

LangGraph Studio 允许您查看、编辑和更新您的助手，并允许您使用这些助手配置运行您的图。

## 图模式 (Graph mode)

要查看您的助手，请单击左下角的 "Manage Assistants"（管理助手）按钮。

这将打开一个模态框，供您查看所选图的所有助手。指定您要标记为 "Active"（活动）的助手及其版本，此助手将在提交运行时使用。

默认情况下，"Default configuration"（默认配置）选项将处于活动状态。此选项反映了图中定义的默认配置。对此配置所做的编辑将用于更新运行时配置，但除非您单击 "Create new assistant"（创建新助手），否则不会更新或创建新助手。

## 聊天模式 (Chat mode)

聊天模式使您能够通过页面顶部的下拉选择器切换图中的不同助手。要创建、编辑或删除助手，请使用图模式。
