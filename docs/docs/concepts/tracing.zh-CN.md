# 追踪 (Tracing)

追踪是您的应用程序从输入到输出所采取的一系列步骤。这些步骤中的每一个都由一次运行表示。您可以使用 [LangSmith](https://smith.langchain.com/) 来可视化这些执行步骤。要使用它，[为您的应用程序启用追踪](../how-tos/enable-tracing.md)。这使您能够执行以下操作：

- [调试本地运行的应用程序](../cloud/how-tos/clone_traces_studio.md)。
- [评估应用程序性能](../agents/evals.md)。
- [监控应用程序](https://docs.smith.langchain.com/observability/how_to_guides/dashboards)。

要开始使用，请在 [LangSmith](https://smith.langchain.com/) 上注册一个免费帐户。
